# Laravel Professional API Auth

A clean, service-oriented authentication package for Laravel 12 using Sanctum.

## Features
- ✅ Registration & Login (Sanctum based)
- ✅ Password Reset (API ready)
- ✅ Email Verification
- ✅ Service-Pattern Architecture (Easy to extend)

## Installation

1. Install the package via composer:
```bash
composer require your-username/laravel-api-auth
```
2. Install filles in your project:
```bash
php artisan api-auth:install
```
